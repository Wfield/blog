## 2021/4/16 明源云  
**1. 下面代码输出什么？**
```
var a = 3
function fn(a) {
    a = 4;
}
fn(a);
console.log(a);
```
输出：3
```
var obj = {a: 3}
function fn(obj) {
    obj.a = 4;
}
fn(obj);
console.log(obj.a);
```
输出：4
